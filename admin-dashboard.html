Como Gerenciar Imagens dos Produtos

Guia rápido para adicionar imagens aos seus produtos

🚨 ERRO ATIVO: QuotaExceededError

SOLUÇÃO IMEDIATA: Execute no console (F12) o comando: corrigirQuotaAgora()

📸 Como Adicionar Imagens aos Produtos:

  1. Vá para a seção "Produtos" no menu lateral
  2. Clique no ícone verde 📷 do produto que deseja editar
  3. No modal que abrir, clique em "Selecionar Imagens"
  4. Escolha suas fotos (JPG, PNG, etc. - máximo 5MB cada)
  5. As imagens aparecerão automaticamente na loja!
